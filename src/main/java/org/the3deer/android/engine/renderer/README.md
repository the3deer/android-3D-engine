# Renderer

The Renderer is responsible for handling the OpenGL onDrawFrame() event.

# Default Renderer

The Default Renderer holds a collection of Drawer that will draw something on the screen.
The default renderer is currently linked to the animation and world matrix update (final solution is still to be developed)  

# Anaglyph Renderer

The Anaglyph renderer is a specialized Stereoscopic renderer capable of rendering both left and right eye.
